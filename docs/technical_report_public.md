# SEIN Industrial Nodal Risk Intelligence - Public Technical Report

This public report summarizes the product methodology, decision value, and limitations. The full processing pipeline and complete analytical dataset remain private.

See the internal/public source version at `docs/technical_report_public.md` in the private project workspace.

## Core Claim

The product is an audited decision-support system for industrial exposure to nodal electricity price stress in Peru. It supports due-diligence prioritization; it is not a causal grid diagnosis, forward price model, billing estimate, or engineering validation layer.
