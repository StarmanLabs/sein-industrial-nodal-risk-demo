# Semantica de Cobertura: Fuente COES vs Score Analitico

El demo publico separa dos ideas que no deben mezclarse:

- `meses_fuente_coes`: trazabilidad de fuente, llave o archivo COES.
- `meses_score`: meses efectivos usados por las senales publicas de estres nodal, prioridad operativa y score de revision.

Una barra puede aparecer con cobertura de fuente durante el periodo 2023-2025 y, aun asi, tener menos meses efectivos de score si los datos disponibles no cumplen el criterio analitico necesario para construir una senal comparable.

Esta distincion evita sobreinterpretar el panel. El producto ordena una cola de revision para due diligence industrial; no prueba congestion fisica, no predice precios, no calcula facturas reales y no reemplaza revision contractual, operativa o de ingenieria.

Regla de lectura:

- Para persistencia, estabilidad y lectura estadistica, usar `meses_score`.
- Para trazabilidad de datos COES, usar `meses_fuente_coes`.
- Si ambos difieren, la barra puede seguir siendo util como caso de screening, pero no debe venderse como senal temporal persistente.
